# Screen Structure And Wireframes

## 1. 전체 화면 구조

```text
앱 시작
  -> 회원가입 / 로그인
  -> 메인 홈
      -> 일반 채팅
          -> 친구 목록
          -> 채팅방 목록
          -> 1:1 채팅방
      -> 랜덤 채팅
          -> 인원 선택
          -> 매칭 대기
          -> 익명 채팅방
      -> 프로필
      -> 테마 설정
      -> 신고 / 차단 관련 진입점
```

## 2. 화면 목록

### 2.1 로그인 화면

목적:

- 기존 사용자 로그인
- 회원가입 화면으로 이동

주요 요소:

- 로고
- 이메일 입력
- 비밀번호 입력
- 로그인 버튼
- 회원가입 버튼

와이어프레임:

```text
+--------------------------------------------------+
|                     LOGO                         |
|         간단한 소개 문구 / 서비스 설명          |
|                                                  |
|  이메일  [_______________________________]       |
|  비밀번호 [_______________________________]      |
|                                                  |
|  [ 로그인 ]                                      |
|  [ 회원가입으로 이동 ]                           |
+--------------------------------------------------+
```

### 2.2 회원가입 화면

목적:

- 신규 사용자 가입

주요 요소:

- 이메일
- 비밀번호
- 비밀번호 확인
- 표시 이름
- 가입 버튼

와이어프레임:

```text
+--------------------------------------------------+
|                   회원가입                       |
|                                                  |
|  이메일        [__________________________]      |
|  비밀번호      [__________________________]      |
|  비밀번호 확인 [__________________________]      |
|  표시 이름     [__________________________]      |
|                                                  |
|  [ 가입하기 ]                                    |
+--------------------------------------------------+
```

### 2.3 메인 홈

목적:

- 일반 채팅 / 랜덤 채팅 진입 분기

주요 요소:

- 상단 프로필 요약
- 큰 아이콘 2개
- 최근 대화 미리보기
- 테마 설정 진입

와이어프레임:

```text
+--------------------------------------------------+
| 프로필 썸네일  안녕하세요, 사용자명              |
| 테마: Midnight Blue                  [설정]      |
|--------------------------------------------------|
|                                                  |
|   [ 랜덤 채팅 아이콘 ]   [ 일반 채팅 아이콘 ]    |
|      랜덤 매칭 시작          친구와 대화         |
|                                                  |
|--------------------------------------------------|
| 최근 대화                                         |
| - 채팅방 A                                        |
| - 채팅방 B                                        |
+--------------------------------------------------+
```

### 2.4 일반 채팅 탭

목적:

- 친구 목록과 기존 채팅방 접근

주요 요소:

- 친구 목록
- 검색
- 채팅방 목록
- 새 대화 시작 버튼

와이어프레임:

```text
+----------------------+---------------------------+
| 친구 목록            | 채팅방 목록               |
| [검색__________]     |                           |
|                      |  채팅방 1                 |
| - 친구 A             |  최근 메시지 미리보기     |
| - 친구 B             |                           |
| - 친구 C             |  채팅방 2                 |
|                      |  최근 메시지 미리보기     |
| [ 새 대화 시작 ]     |                           |
+----------------------+---------------------------+
```

### 2.5 일반 1:1 채팅방

목적:

- 친구와 텍스트 대화

주요 요소:

- 상대 이름
- 신고 / 차단 메뉴
- 메시지 목록
- 입력창
- 읽음 상태
- 비활성 버튼: 파일, 통화, 선물

와이어프레임:

```text
+--------------------------------------------------+
| < 뒤로   친구 A                         [메뉴]   |
|--------------------------------------------------|
|  상대: 안녕?                                     |
|                          나: 반가워              |
|  상대: 오늘 시간 돼?                              |
|                          나: 응 가능             |
|                                                  |
|                    읽음 표시                      |
|--------------------------------------------------|
| [파일] [통화] [선물]  [메시지 입력__________] [전송] |
+--------------------------------------------------+
```

메모:

- MVP에서는 파일/통화/선물 버튼은 비활성 또는 `준비 중` 처리

### 2.6 랜덤 채팅 설정 화면

목적:

- 랜덤 채팅 시작 전 기본 조건 설정

주요 요소:

- 인원 수 선택
- 시작 버튼
- 주의 문구

와이어프레임:

```text
+--------------------------------------------------+
|                  랜덤 채팅 시작                  |
|                                                  |
|  인원 선택                                       |
|  ( ) 1:1   ( ) 2:2   ( ) 3명   ( ) 4명          |
|                                                  |
|  안내: 개인정보를 공유하지 마세요.               |
|  파일, 선물, 통화는 지원하지 않습니다.            |
|                                                  |
|  [ 매칭 시작 ]                                   |
+--------------------------------------------------+
```

메모:

- MVP에서는 실제 동작은 `1:1`만 활성화

### 2.7 랜덤 매칭 대기 화면

목적:

- 대기 상태 표시

주요 요소:

- 매칭 중 애니메이션 또는 상태 표시
- 취소 버튼

와이어프레임:

```text
+--------------------------------------------------+
|                매칭 상대를 찾는 중...            |
|                                                  |
|                  [ 로딩 표시 ]                   |
|                                                  |
|      익명 닉네임이 자동 생성되어 입장합니다      |
|                                                  |
|                  [ 취소 ]                        |
+--------------------------------------------------+
```

### 2.8 랜덤 채팅방

목적:

- 익명 대화 진행

주요 요소:

- 상단 익명 닉네임 표시
- 신고 / 나가기 버튼
- 메시지 목록
- 입력창
- 금지 기능 비노출

와이어프레임:

```text
+--------------------------------------------------+
| < 뒤로   별빛고양이 / 상대: 구름호랑이   [신고]  |
|--------------------------------------------------|
|  구름호랑이: 안녕하세요                           |
|                           별빛고양이: 안녕하세요  |
|  구름호랑이: 어디 사세요?                         |
|                           시스템: 개인정보 공유 주의 |
|                                                  |
|--------------------------------------------------|
|  [메시지 입력____________________________] [전송] |
+--------------------------------------------------+
```

메모:

- 프로필 이미지, 실명, 파일 버튼, 선물 버튼, 통화 버튼 없음

### 2.9 프로필 / 설정 화면

목적:

- 표시 이름, 소개, 테마 설정

주요 요소:

- 프로필 이미지
- 표시 이름
- 소개
- 테마 선택
- 저장 버튼

와이어프레임:

```text
+--------------------------------------------------+
|                    프로필 설정                    |
|                                                  |
|  프로필 이미지  [변경]                           |
|  표시 이름     [__________________________]      |
|  소개          [__________________________]      |
|                                                  |
|  테마 선택                                       |
|  [ Light ] [ Dark ] [ Neon ] [ Custom ]          |
|                                                  |
|  [ 저장 ]                                        |
+--------------------------------------------------+
```

### 2.10 신고 화면

목적:

- 사용자 또는 메시지 신고

주요 요소:

- 신고 대상
- 신고 사유 선택
- 상세 입력
- 제출 버튼

와이어프레임:

```text
+--------------------------------------------------+
|                     신고하기                      |
|                                                  |
|  대상: 구름호랑이                                |
|                                                  |
|  사유                                             |
|  ( ) 욕설  ( ) 스팸  ( ) 성희롱  ( ) 기타        |
|                                                  |
|  상세 내용                                        |
|  [__________________________________________]    |
|                                                  |
|  [ 신고 제출 ]                                   |
+--------------------------------------------------+
```

## 3. 내비게이션 구조 제안

- 하단 탭 또는 상단 메인 메뉴 기준
- `홈`
- `일반 채팅`
- `랜덤 채팅`
- `프로필`

## 4. MVP 우선순위

1. 로그인 / 회원가입
2. 홈에서 두 채팅 타입 분기
3. 일반 1:1 채팅
4. 랜덤 1:1 채팅
5. 신고 / 차단
6. 테마 적용

## 5. UI 정책 메모

- 일반 채팅과 랜덤 채팅은 색상 또는 배지로 명확히 구분
- 랜덤 채팅방에는 익명성 보호 안내를 반복 노출
- 금지 기능은 숨기거나 비활성화해 오해를 줄임
- 테마는 채팅 가독성을 해치지 않는 범위에서만 허용
